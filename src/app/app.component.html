Welcome to Samsung Electro-Mechanics UI/UX Developer Coding Challenge!

To begin, kindly replace the "Candidate" with your name.

Implement the following...

  1. Create one parent and one child component...
    • In child component, create a an input field and a button. Display the entered value in the input field in the parent component.
      (Bonus points if you use form validation using Reactive Forms
  2. Create a list of your favorite items (Sports, chocolates, fruits, etc...) and display in template using *ngFor directive.
  3. Show an example using date pipe.
    (Extra cookie points if you use arguments for the pipe).
  4. Create a custom pipe to trim a string with more than 5 characters.
  5. Add a Color Picker field and use Two-Way Data Binding to change the background color of a div
